EXERCICE 1 : Premier pointeur
Crée une variable int age = 18, puis un pointeur vers cette variable.
Affiche l'adresse et la valeur via le pointeur.
Objectif : Comprendre & et *
[ ] Exercice terminé

EXERCICE 2 : Modifier via pointeur
Crée une variable float prix = 19.99.
Via un pointeur, change le prix à 24.99 et affiche le résultat.
Objectif : Modifier une valeur via déréférencement
[ ] Exercice terminé

EXERCICE 3 : Fonction doubler
Écris une fonction qui double la valeur d'un int via un pointeur.
Teste avec plusieurs valeurs.
Objectif : Passer un pointeur à une fonction
[ ] Exercice terminé

EXERCICE 4 : Pointeur NULL
Déclare un pointeur int* non initialisé, puis initialise-le à NULL.
Vérifie qu'il est NULL avant de l'utiliser.
Objectif : Éviter les erreurs de segmentation
[ ] Exercice terminé

EXERCICE 5 : Comparer des adresses
Crée deux variables int, affiche leurs adresses.
Détermine quelle variable est "plus haute" en mémoire.
Objectif : Comprendre l'organisation de la stack
[ ] Exercice terminé

EXERCICE 6 : Pointeurs de différents types
Crée des pointeurs pour char, int, float, double.
Affiche la taille de chaque pointeur avec sizeof().
Objectif : Voir que tous les pointeurs ont la même taille
[ ] Exercice terminé

EXERCICE 7 : Fonction min et max
Écris une fonction qui prend deux pointeurs int et affiche le plus petit et le plus grand.
Objectif : Déréférencer plusieurs pointeurs
[ ] Exercice terminé

EXERCICE 8 : Pointeur sur tableau
Crée un tableau int tab[3] = {10, 20, 30}.
Utilise un pointeur pour afficher chaque élément.
Objectif : Comprendre que le nom d'un tableau est un pointeur
[ ] Exercice terminé
