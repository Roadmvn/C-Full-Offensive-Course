# EXERCICE : ARBRES BINAIRES


### OBJECTIF :
Implémenter un Binary Search Tree (BST) complet avec opérations avancées.

FONCTIONS À IMPLÉMENTER :

1. insert(root, value) - Insérer un nœud
2. search(root, value) - Rechercher un nœud
3. delete_node(root, value) - Supprimer un nœud
4. find_min(root) - Trouver le minimum
5. find_max(root) - Trouver le maximum
6. height(root) - Calculer la hauteur
7. count_nodes(root) - Compter les nœuds
8. count_leaves(root) - Compter les feuilles
9. is_bst(root) - Vérifier si c'est un BST valide
10. inorder(root) - Parcours in-order
11. preorder(root) - Parcours pre-order
12. postorder(root) - Parcours post-order
13. levelorder(root) - Parcours par niveau
14. mirror_tree(root) - Créer le miroir
15. find_lca(root, n1, n2) - Plus petit ancêtre commun
16. is_balanced(root) - Vérifier si équilibré
17. print_tree(root) - Affichage visuel
18. free_tree(root) - Libérer la mémoire


### EXERCICES BONUS :

- Convertir un tableau trié en BST équilibré
- Vérifier si deux arbres sont identiques
- Calculer le diamètre de l'arbre
- Trouver tous les chemins racine → feuille
- Sérialiser/Désérialiser un arbre

FICHIER : main.c

**COMPILATION : gcc main.c -o bst -Wall**
